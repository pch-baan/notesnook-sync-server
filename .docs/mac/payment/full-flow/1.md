```

║                                                              ║
║  ① Xem Webhook Notifications:                               ║
║  ───────────────────────────                                 ║
║  Developer Tools → Notifications                            ║
║  → Click "Notesnook Sync Server"                            ║
║  → Tab "Events" / "Notifications"                           ║
║  → Thấy danh sách events đã gửi:                           ║
║     • transaction.completed    → delivered ✅               ║
║     • subscription.created     → delivered ✅               ║
║     • subscription.activated   → delivered ✅               ║
║     • customer.created         → delivered ✅               ║

║  ② Xem Subscriptions:                                       ║
```
